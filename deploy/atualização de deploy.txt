Atualização de Deploy

Commit das Alterações no Repositório:

git add .

git commit -m "Atualização do projeto"

git push origin main

npm run deploy

Vá até o repositório no GitHub:
Clique em Settings > Pages e confirme o status do deploy
Teste o Site Online:

Acesse o link gerado pelo GitHub Pages (algo como https://seu-usuario.github.io/nome-do-repositorio).
